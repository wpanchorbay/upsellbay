## Offer Form Configuration

Clicking **Add Offer** opens the custom configuration page.

### Configuration Fields

- **Title:** Internal administrative name.
- **Offered Product:** Select simple or variable WooCommerce products.
- **Discount settings:** Set fixed amounts, percentage deductions, or fixed pricing.
- **Placement Hook:** Choose where to inject the offer layout.
- **Targeting Criteria:** Define standard AND/OR rule groups.
