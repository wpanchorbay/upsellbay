## Technical Options

Tune database execution and scheduler frequencies under Advanced Settings.

- **Action Scheduler Intervals:** Set daily rollups processing rate.
- **Transient Duration:** Set duration rules for eligibility caching.
- **Log Level:** Choose diagnostic levels (Errors, Warnings, Full Debug).
