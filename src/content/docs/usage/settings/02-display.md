## Design Customization

Visual settings adjust how elements render on single product templates, carts, and checkout checkboxes.

- **Inherit Theme Styles:** Checkbox to pull font families, sizing, and basic input styling from active themes.
- **Color Settings:** Accent colors, boundaries, and button layouts.
- **Keyboard Traps:** Rules ensuring screen readers and tab navigation work with storefront toggles.
