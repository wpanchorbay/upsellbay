## Checkout Integrations

### Classic Checkout

Injected above the place order button via `woocommerce_review_order_before_submit`.

### Block Checkout

Integrated using WooCommerce's Additional Checkout Fields API. Displays a checklist card within the checkout block framework.

## Accessibility

All checkout toggles support screen readers, label associations, and keyboard navigation.
