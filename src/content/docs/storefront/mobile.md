## Mobile Styling Guidelines

Storefront offer cards automatically adapt to mobile screen widths:

- **Touch Target Padding:** Inputs and buttons have a minimum size of 44x44 pixels.
- **Adaptive Grid Layouts:** Horizontal grids transition to single column structures on screen widths below 768px.
- **Overflow Limits:** CSS prevents horizontal page scrolls and bounds text labels.
