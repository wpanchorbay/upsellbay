## Compatibility Approach

UpsellBay is built to be WooCommerce-native, but every store has its own theme, checkout customizations, and plugin stack. The safest rollout path is always staged validation.

## Areas To Validate

- theme styling on product, cart, checkout, and thank-you pages
- any checkout-field or checkout-layout plugins
- cart drawer or mini-cart behavior
- product types used as offer products
- mobile layout behavior
