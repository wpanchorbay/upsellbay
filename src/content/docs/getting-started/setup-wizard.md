## Onboarding Walkthrough

Upon first activation, UpsellBay launches an onboarding setup wizard to configure baseline parameters.

### Wizard Steps

1. **License Key:** Enter your premium license key to enable background updates and diagnostics.
2. **Default Styling:** Choose default styling matching your active theme (colors, borders, typography).
3. **Basic Analytics:** Enable non-PII aggregate analytics for performance reports.
4. **Test Mode:** Choose whether to keep test mode active for initially designed offers.

## Skipping the Wizard

The wizard can be bypassed at any point. Manual options remain editable under **WooCommerce > UpsellBay > Settings**.
