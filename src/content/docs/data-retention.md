## What UpsellBay Stores

- offer definitions
- global plugin settings
- aggregate non-PII analytics
- temporary offer session state
- operational logs

## Default Behavior

UpsellBay preserves data by default on uninstall unless the merchant explicitly enables uninstall cleanup.
